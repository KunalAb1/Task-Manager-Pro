# Task Manager Pro

A full-stack task manager built with Node.js, Express and vanilla JavaScript.

## Features
- Add, edit, delete tasks
- Task completion toggle
- Due dates with overdue detection
- Priority levels (High/Medium/Low)
- Filter tasks (All/Active/Completed/High)
- Task counter
- JSON file persistence

## How to run
1. Clone the repo
2. Run `npm install`
3. Run `node index.js`
4. Open `index.html` in browser
